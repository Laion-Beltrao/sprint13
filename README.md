# sprint13

## Visão Geral
Neste projeto, foi feita uma análise completa de churn (cancelamento de clientes) para uma rede de academias, com foco em:
- Identificar fatores que levam à evasão
- Construir modelos preditivos de churn
- Segmentar clientes com base em comportamento
- Gerar recomendações estratégicas de retenção

## Problema de Negócio
A academia enfrenta perda significativa de clientes e precisa responder:

- Quais clientes têm maior risco de churn?
- Quais comportamentos indicam abandono iminente?
- Como reduzir a taxa de cancelamento?

## Abordagem Analítica
Análise exploratória (EDA)
Comparação entre clientes ativos vs churn
Modelos preditivos:
Regressão Logística
Random Forest
Segmentação de clientes (K-Means)
Análise de churn por cluster
